# musegit

Private collaboration for indie bands and remote music projects.

`musegit` is a web-first workspace for songs, mixes, and review comments. The initial product focuses on a simple problem: helping collaborators know which file is current, what changed, and whether a version is approved.

## MVP Focus

- Private band workspaces
- Songs and mixes as the core unit of work
- Versioned asset uploads
- Comments and review state on each version
- Web-only first

## Product Direction

This is not a public music platform and not a DAW replacement.

The first version is aimed at:

- indie bands
- remote collaborators
- small music teams currently using shared folders and chat
